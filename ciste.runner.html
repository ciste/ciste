ciste.runner documentation

This is the runner for ciste applications.

Specify this namespace as the main class of your application.

-main

(-main & options)
Main entry point for ciste applications.

Specify this function as you application's entry point.

default-site-config

Ref containing the currently loaded site config

default-site-config-filename

By default, the runner will look for a file with this name at the root
of the project directory.

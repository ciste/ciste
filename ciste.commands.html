ciste.commands documentation

*command-predicates*
The sequence of predicates used for command dispatch.
By default, commands are dispatched by name.
*commands*
The sequence of commands that have been registered.

add-command!
(add-command! name v)
Adds the fn identified by var v as the command handler for the given name.

command-names
(command-names)
The names of all the registered commands.

parse-command
(parse-command {:as command})
Takes a sequence of key/value pairs and runs a command
